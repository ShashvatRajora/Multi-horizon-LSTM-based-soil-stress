Comprehensive Data Summary and Usage in Project
1. Overview

The project utilizes a multimodal dataset constructed from multiple heterogeneous sources to simulate a realistic agricultural IoT ecosystem. The collected data spans environmental, soil, and temporal dimensions, enabling predictive modeling of crop stress and disease.

The datasets have been carefully selected and combined to ensure:

Temporal continuity (time-series learning)
Environmental diversity (multi-location climate data)
Soil variability (physical and chemical properties)
Realistic IoT simulation

🧠 8. How the Data Works Together

The system integrates all datasets into a unified time-series pipeline:

NASA data → macro-level climate patterns
IoT datasets → micro-level sensor variations
Soil dataset → static physical properties
Moisture dataset → temporal soil behavior

👉 Together, they simulate a complete agricultural ecosystem

🔮 9. Final Use in Model

The combined dataset is used to:

Input:
Time-series sequences (past 24–72 timesteps)
Leaf images (from FarmBud)
Output:
Future disease probability
Stress classification
Recommendations
